Приложение "Delivery Service" ("Служба доставки") может быть полезно курьерским и подобным службам. Приложение позволяет рассчитать оптимальный порядок посещения заданных точек с учетом их географического расположения и желаемого времени их посещения.

Адреса доставки можно вводить вручную, указывать двойным щелчком по карте либо взять из доступных геозон и POI. Приложение позволяет задать любое количество точек и указать для каждой из них временные рамки ее посещения.

Расчет маршрута может производиться по картам Yandex, с учетом дорог либо просто по прямой. В результате расчета устанавливается оптимальный порядок посещения точек, и предлагаемый маршрут движения показывается на карте. Подробная информация по полученному маршруту выводится также в виде таблицы, с последовательным указанием всех адресов, времени их посещения и расстояния между точками.

Рассчитанный маршрут можно отправить на печать. Кроме того, его можно сохранить в системе Wialon Hosting и использовать в дальнейшем для различных целей мониторинга.
